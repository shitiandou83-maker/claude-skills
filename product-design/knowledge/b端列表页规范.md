# DongDesign B端列表页规范

本文档的内容为B端列表页通用场景下必须遵守的默认规则，请在页面搭建时务必遵守。

## 一、组件构成

B端列表页的搭建必须由 dongDesign-B 组件库中的组件搭建，组件定义如下：

- [Cascader 级联选择器]
- [Checkbox 多选框]
- [DatePicker 日期选择器]
- [Input 输入框]
- [Radio 单选框]
- [Select 选择器]
- [TimePicker 时间选择器]
- [Divider 分割线]
- [Upload 上传]
- [Table 表格]
- [Button 按钮]
- [Pagnation 分页]
- [Tabs 标签页]

## 二、基础样式

### 框架布局

#### 设计尺寸
- 一般情况下，后台系统界面的设计尺寸为 [长：1440px，高：900px]

#### bg-color
- 背景色值为 [surface-100]

#### 顶导航
- 高度 [48px]，宽度撑满页面，位于页面顶部
- 由 logoArea（必有）、menuArea（可有）、UserInfo（必有）构成
- 使用水平居中布局，左右两侧 padding 为 [20px]，区域之间间距为 [0px]
- 可使用顶导航组件 [ProTopMenu 顶部菜单]

#### 左导航
- 宽度 [200px]，高度撑满页面，位于顶导航下方，页面左侧
- 一般情况下可使用顶导航组件 [ProSingleMenu 单列菜单]
- 当需要支持多级菜单展示、收藏和插槽等扩展能力时，可使用 [ProMenu 双栏菜单]

#### 页面信息区
- 高度及构成：高度 [28px]，由面包屑和扩展操作区构成，使用水平居中布局，位于顶导航下方，左导航右侧，内容卡片上方
- 面包屑：必有，居左布局，使用组件 [ProTopMenu 面包屑]
- 扩展操作区：可有，居右布局，展示页面负责人信息，使用组件 [ProErpButton Erp按钮]
  - 按钮之间需用分割线分割，分割线宽 [1px]，高 [14px]，色值为 [line-100]
  - 分割线到两侧按钮之间间距为 [8px]
  - 此处仅展示页面负责人信息，不允许出现其他页面内的操作按钮

#### 页面内容区
- 卡片容器：内容区智能由一个白色卡片容器包裹，卡片需撑满整个页面，距离页面右侧边距为 [16px]
  - 卡片背景色值为 [fill-white]
  - 上方左右两个圆角半径为 [8px]，下方左右两个圆角半径为 [0px]
  - 卡片左右两侧 padding 为 [24px]，上方 padding 为 [20px]
  - 内容展示不下时卡片内出纵向滚动条，且滚动到底部时页面内内容底部距离页面底部需预留高度为 [80px] 的留白区

#### 间距
- 顶导航 → 左导航：间距 0px
- 顶导航 → 页面信息区：间距 10px
- 左导航 → 页面信息区：间距 0px
- 左导航 → 页面内容区：间距 0px
- 页面信息区 → 页面内容区：间距 4px

## 三、页面内容区布局

页面内容区的布局需要区分是否存在操作按钮区：
- 无操作按钮区时，数据查询区直接跟表格容器相邻
- 有操作按钮区时，数据查询区域跟操作按钮区之间需要有分割线分割

### 场景一：无操作按钮区

#### 页面标题
- 页面名称，与页面内容左对齐，使用 dongDesign-B React 的基础组件组件库中标题组件 [h1.dongDesign 页面标题]

#### 数据查询
- 数据查询使用 [Form 表单] 中的高级搜索，即三列栅格式的表单排列方式
- 查询项必须包含 label，且 label 宽度按照标题最长的标题字符数所需宽度预留
- 数据查询必须展示查询按钮组，查询按钮组必须包含查询、重置两个操作
  - 查询按钮为主要操作，使用线性主要按钮样式
  - 重置为次要操作，使用线性次要按钮样式
- 当数据查询包含的查询项较多时（超过五个），可增加展开收起操作
  - 使用文本按钮 + 后置图标样式
  - 默认为收起状态显示展开操作，后置图标为 `<DownOutlined/>`
  - 展开后操作变为收起，后置图标变为 `<UpOutlined/>`
- 为保证按钮跟同列查询项左对齐，查询按钮组也需预留 Label 宽度，但无需展示标题
- 查询项小于等于两项时，查询按钮组跟随在查询项后侧，间距为 24px

#### 表格布局
- **表格容器（必须遵守）**：表格容器必须存在 1px 的 border，色值为 [line-100]，圆角半径为 4px，容器需裁切内容，即表格容器的 enableLegacyStyleNoOuterBorder 属性值为 true
- **内容列**：从表格容器左侧开始水平布局排列，列间距为 0px，宽度设置需考虑按列内容和列表整体的展示效果，列内容存在截断时用"..."代替截断内容，鼠标 hover 至对应截短内容时出现文字提示组件 [Tooltip 文字提示] 展示完整内容，组件需指向对应内容
- **状态列**：当存在状态列时，可在单元格内使用 [Tag 标签] 的 vivid 类型
  - 状态颜色：进行中 [蓝(主色)]，失效或完成状态 [灰]，待处理状态 [橙]，已通过 [绿]，异常状态 [红]
- **操作列**：位于表格最右侧，使用 [Button 按钮] 中的链接按钮样式，不需要展示图标，操作按钮之间间距为 [8px]
- **表格适配**：列表内容列展示不下时，操作列吸附在右侧固定展示，并增加投影 [Shadow-middle]

#### 表格分页
- 使用分页组件 [Pagnation 分页]，位于表格容器下方，居右布局，与表格容器右对齐

#### 间距
- 页面标题 → 数据查询：间距 20px
- 数据查询 → 表格容器：间距 24px
- 表格容器 → 分页器：间距 16px

### 场景二：有操作按钮区

#### 页面标题
- 页面名称，与页面内容左对齐，使用 dongDesign-B React 的基础组件组件库中标题组件 [h1.dongDesign 页面标题]

#### 数据查询
- 数据查询使用 [Form 表单] 中的高级搜索，即三列栅格式的表单排列方式
- 查询项必须包含 label，且 label 宽度按照标题最长的标题字符数所需宽度预留
- 数据查询必须展示查询按钮组，查询按钮组必须包含查询、重置两个操作
  - 查询按钮为主要操作，使用线性主要按钮样式
  - 重置为次要操作，使用线性次要按钮样式
- 当数据查询包含的查询项较多时（超过五个），可增加展开收起操作
  - 使用文本按钮 + 后置图标样式
  - 默认为收起状态显示展开操作，后置图标为 `<DownOutlined/>`
  - 展开后操作变为收起，后置图标变为 `<UpOutlined/>`
- 为保证按钮跟同列查询项左对齐，查询按钮组也需预留 Label 宽度，但无需展示标题
- 查询项小于等于两项时，查询按钮组跟随在查询项后侧，间距为 24px

#### 分割线
- 数据查询区跟按钮操作区之间需要有分割线隔开，使用分割线组件 [Divider/水平]
- 分割线与上下两个区域之间的间距为 16px

#### 操作按钮区

操作按钮区由数据操作类按钮组和展示设置类按钮组构成，数据操作类按钮组居左，展示设置类按钮组居右，使用水平居中对齐布局。

**展示设置类按钮组**：
- 位于数据表格上方与页面标题和表格容器右对齐水平排列
- 按钮参数为 [Button按钮/中/次要/链接/前缀图标]
- 按钮之间间距为 [12px]

**数据操作类按钮组**：
- 位于数据表格上方与页面标题和表格容器左对齐水平排列
- 分为主要操作按钮和次要操作按钮，按钮之间间距为 [8px]
- 可使用按钮组组件 [Button Group]
- **主要操作按钮**：位于操作按钮组最左侧（如"+新建商品"按钮），使用按钮组件 [Button/中/主要/填充/前缀图标 plus]
- **次要操作按钮**：排在主要操作按钮右侧（如"批量上架"、"批量下架"），使用按钮组件 [Button/中/次要/线性]
- **依赖表格数据勾选的批量操作按钮**：需区分已勾选状态和未勾选状态
  - 未勾选时该批量操作为禁用态，使用按钮组件 [Button/中/次要/线性/禁用]
  - 鼠标移入区域时 hover 出现 tooltip 组件 [Tooltip/大/浅色] 提示相关信息"勾选列表数据后可操作"
- 存在依赖表格数据勾选的批量操作时，需在按钮组左侧展示勾选数据信息，距离按钮组之间间距为 [8px]
  - 无勾选数据时，信息展示格式为：共 {count} 条，各元素之间间距为 4px
  - 其中「共」「条」色值为 text-200，{count} 色值为 text-400，字重为 medium，展示数值为当前页面展示数据条数

#### 表格布局
- **表格容器（必须遵守）**：表格容器存在 1px 的 border，色值为 [line-100]，圆角半径为 4px，容器需裁切内容
- **多选框列**：位于列表的最左侧，单元格内为 [Checkbox 多选框]，且不展示文字
- **内容列**：从表格容器左侧开始水平布局排列，列间距为 0px，宽度设置需考虑按列内容和列表整体的展示效果，列内容存在截断时用"..."代替截断内容，鼠标 hover 至对应截短内容时出现文字提示组件 [Tooltip 文字提示] 展示完整内容，组件需指向对应内容
- **状态列**：当存在状态列时，可在单元格内使用 [Tag 标签] 的 vivid 类型
  - 状态颜色：进行中 [蓝(主色)]，失效或完成状态 [灰]，待处理状态 [橙]，已通过 [绿]，异常状态 [红]
- **操作列**：位于表格最右侧，使用 [Button 按钮] 中的 type 属性值为 link，不需要展示图标，操作按钮之间间距为 [8px]
- **表格适配**：列表内容列展示不下时，多选框列吸附在左侧固定展示，操作列吸附在右侧固定展示，并增加投影 [Shadow-middle]
- **内容行高**：表格列存在主次文案时，行高随行内高度最大的单元格高度撑开，其他单元格高度需撑满行高
- **可设置主次文案的表格列**：
  - 商品信息 — 参数为 [Table表格列/商品信息/辅助信息(ID)]
  - 主次文本 — 参数为 [Table表格列/主次文本/1行描述]
  - 数字 — 参数为 [Table表格列/数字/主数据币值符/主数据小数点/1行子指标]

#### 表格分页
- 使用分页组件 [Pagnation 分页]，位于表格容器下方，居右布局，与表格容器右对齐

#### 间距
- 页面标题 → 数据查询：间距 20px
- 数据查询 → 分割线：间距 16px
- 分割线 → 表格容器：间距 16px
- 表格容器 → 分页器：间距 16px
